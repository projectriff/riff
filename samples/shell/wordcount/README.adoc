= wordcount

[horizontal]
*Language*:: Shell
*Function*:: https://github.com/projectriff/riff/blob/master/samples/shell/wordcount/wordcount.sh[wordcount.sh]
*Protocol*:: stdio
*Input*:: names
*Output*:: greetings

=== Build the Docker container for the function code

```
./riff build -n wordcount -v v0001 -f samples/shell/wordcount
```

=== Create the Function and Topic(s)

```
./riff apply -f samples/shell/wordcount
```

=== Publish a Message to the Input Topic

```
./publish names hello
```

=== Tail the Sidecar Log

```
./riff logs -n wordcount -t
```

=== Delete the function

```
./riff delete -n wordcount
```
